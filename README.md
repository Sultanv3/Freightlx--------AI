# FREIGHTLX

منصة ذكية للشحن الدولي والتخليص الجمركي وسابر، مدعومة بالذكاء الاصطناعي.

## الملفات

- `index.html` — التطبيق الرئيسي (Single Page Application)
- `assets/logo/` — شعار FREIGHTLX الرسمي
- `data/` — ملفات البيانات المرجعية
- `vercel.json` — إعدادات النشر على Vercel
- `robots.txt`, `sitemap.xml` — SEO

## تشغيل محلي

```bash
npm start
```

ثم افتح http://localhost:3000

## النشر

راجع `DEPLOYMENT-GUIDE.md` لخطوات النشر الكاملة بالعربي.
